Déployez simplement votre outil de consultation en ligne

1. Sélectionnez l'outil

Étudiez attentivement la documentation mise à disposition par les éditeurs de logiciel et consultez les cas d'usage. N'hésitez pas à nous contacter ou à contacter les éditeurs. Vous pouvez aussi déployer gratuitement les outils pour les tester.

2. Remplissez le formulaire

Une fois l'outil choisi, rendez-vous sur la page de l'outil et cliquez sur "Utilisez" dans le bandeau en haut de la page.

Définissez l’URL et le nom de votre consultation et entrez vos informations personnelles (nom, prénom, administration). Saisissez votre adresse e-mail. À noter que seules les adresses de l’administration centrale (.gouv.fr) sont acceptées pour l’instant.

Cette démarche vous permet de demander la création, sous 48h, d'une base de données et d'une instance de consultation, hébergées chez l'éditeur de solution, auxquelles vous seul aurez accès.

3. Validez votre adresse e-mail

Vous allez rapidement recevoir un e-mail de confirmation. Il faut bien vérifier que cet e-mail ne soit pas dans vos spams.
Cliquez sur le lien mentionné sur l’e-mail pour valider votre adresse.

4. Créez votre consultation !

L'outil' va mettre en place la base de données de votre consultation. Le délai de déploiement peut prendre jusqu’à 48h.
Vous recevrez un e-mail lorsque la base de données sera prête et l'instance de consultation déployée. Vous pouvez commencer à prendre en main l'outil.

5. Paramétrez votre consultation

Si vous voulez commencer à paramétrer vous-même votre consultation, consultez la documentation disponible.

Si vous voulez faire appel à un prestataire de service pour vous accompagner, consultez la page accompagnement.
